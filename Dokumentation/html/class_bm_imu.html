Detailed Description

Abstraktionsklasse für die Sensor-ICs Bosch BMI270 (IMU) und BMM150 (Magnetometer).

  • Bietet eine standardisierte objektorientierte API zum Auslesen der kinematischen und magnetischen Messdaten auf dem Arduino Nano 33 BLE Sense.

Prüft, ob neue Beschleunigungsdaten im FIFO/Register verfügbar sind.

Überprüft den Hardware-Interrupt-Status auf neue Beschleunigungsdaten (Data Ready).

Returns

int8_t BmImu::configure_sensor ( struct bmi2_dev * dev)
protected

Interne hardwarenahe Konfiguration des BMI270 IMU-ICs.

Interne Konfigurationsfunktion für das BMI270 IMU-IC.

Parameters
devZeiger auf die Bosch-Treiberstruktur der IMU.
Returns
int8_t Statuscode der Bosch-API.
  • Richtet den Interrupt-Pin INT1 (Active High, Push-Pull, Output enabled) ein, setzt die Abtastrate für Beschleunigung und Gyroskop auf 50 Hz, wählt die Arbeitsbereiche ($\pm 4\,\text{g}$ und $\pm 2000\,\text{dps}$) und deaktiviert den Advanced-Power-Save-Modus für stabileres Signalverhalten während transienter Messungen.
  • Parameters
    devZeiger auf die Bosch Treiberstruktur.
    Returns
    int8_t Statuscode der Operation (BMI2_OK bei Erfolg).
