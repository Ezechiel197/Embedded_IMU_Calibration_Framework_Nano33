Abstraktionsklasse für die Sensor-\/\+ICs Bosch BMI270 (IMU) und BMM150 (Magnetometer). 


\begin{DoxyItemize}
\item Bietet eine standardisierte objektorientierte API zum Auslesen der kinematischen und magnetischen Messdaten auf dem Arduino Nano 33 BLE Sense.
